#  Online STEM Book Store

##  Overview
The Online STEM Book Store is a web-based platform designed to serve students, educators, and professionals in the fields of Science, Technology, Engineering, and Mathematics (STEM).

The platform provides a centralized digital marketplace where users can browse, purchase, and sell STEM-related books in multiple formats including printed copies, PDFs, and e-books.

---

##  Objective
To develop a secure and user-friendly online platform that:
- Provides access to STEM textbooks and learning resources
- Supports both new and secondhand book sales
- Enables secure online transactions
- Offers flexible digital access to purchased materials

---

## Tech Stack
- **Frontend:** Python (Django Framework)
- **Backend:** MySQL
- **Web Server:** XAMPP
- **Architecture:** Client-Server Model

---


##  User Roles & Features

###  Admin
- Secure login using username and password
- Manage homepage content
- Register and manage districts, locations, categories, and sub-categories
- View and approve author registrations
- View and approve book registrations
- Manage author details, user details, and sales reports
  

---

###  Author
- Secure login using username and password
- Author self-registration
- Access personal dashboard (homepage)
- Add/register new books
- Update or delete listed books
- View sales and payment details
- Logout securely

---

###  User
- Secure login using username and password
- User self-registration
- Access homepage
- Browse and view book details
- Purchase books (Printed copy / PDF / E-book)
- View order history and payment details
- Register and sell second-hand books
- Logout securely
